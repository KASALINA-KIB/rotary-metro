Registration Portal Project

About the Project

A modern digital registration and attendance management system for the Rotary Club of Kampala Metropolitan. This web-based portal streamlines member registration, guest check-ins, attendance tracking, and makeup logging for rotary meetings.

View Live Site

Key Features

Member Portal

Secure login and registration system for club members with personalized dashboards.

Guest Registration

Easy registration for visiting Rotarians, Rotaractors, and guests with instant QR code access.

Attendance Tracking

Real-time attendance logging with buddy group selection and live statistics.

Makeup Logging

Track member visits to other clubs (up to 7 makeups per period) with automated reporting.

Admin Dashboard

Comprehensive admin controls for viewing records, exporting data (CSV/JSON), and managing attendance.

QR Code Access

Scan-to-register functionality for quick and contactless attendance at events.
